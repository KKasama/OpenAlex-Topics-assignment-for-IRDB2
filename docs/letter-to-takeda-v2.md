## 1. OpenAlex の誤分類の根本原因

OpenAlex の Topic 分類モデルのソースコードを精査いたしました（公開リポジトリ：https://github.com/ourresearch/openalex-topic-classification）。

### モデルの構造

OpenAlex は `bert-base-multilingual-cased`（多言語対応 BERT）を独自にファインチューニングしたモデルを使用しています。理論上は日本語を含む多言語に対応していますが、以下の 3 つの構造的問題により、**日本語論文では機能しない**ことが確認できました。

| 問題点 | 詳細 |
|---|---|
| ① 学習データが英語中心 | 学習データは CWTS（オランダ・ライデン大学）提供の約 430 万件で、欧米英語論文が大半を占めます。日本語論文の割合が極めて少なく、ファインチューニングが英語特化となっています。 |
| ② 引用ネットワーク特徴への依存 | モデルはテキストに加え「引用論文の埋め込み」を入力として使用します。国際引用ネットワークへの参加が少ない日本語論文では、この特徴量が機能せず、誤った方向に引き寄せられます。 |
| ③ 多言語 BERT の能力が活かされない | 多言語 BERT の日本語処理能力は、英語中心のファインチューニングによってほとんど失われています。これが「Military Technology and Strategies」への大量集中の直接的な原因です。 |

### 1 論文あたりの Topic 付与数

| 項目 | 件数 |
|---|---|
| モデルが内部で計算する数 | **上位 10 件**（信頼度スコア付き） |
| OpenAlex API で公開される数 | **通常 3 件**（`topics` 配列） |
| 最高スコアの 1 件 | `primary_topic`（別フィールド） |

## 2. 改良手法 v2（アンサンブル）の概要

前回の手法（v1：埋め込みモデル単体）に加え、**BM25 キーワードマッチング**を組み合わせたアンサンブル手法（v2）を実装・試験いたしました。

```
論文テキスト（タイトル + 要旨）
      │
      ├─【A】multilingual-e5-base（埋め込み）─→ コサイン類似度スコア
      │
      └─【B】BM25（キーワードマッチング）────→ 正規化スコア
                        │
                        ▼
            最終スコア = 0.75 × 埋め込みスコア + 0.25 × BM25スコア
```

BM25 は OpenAlex Topic の `keywords` / `display_name` / `description` との語彙的一致を検出し、埋め込みモデルが苦手な固有表現・技術用語・略語を補完します。

## 3. 1,000 件サンプルによる v1 vs v2 比較

IRDB 論文 1,000 件を対象に v1（埋め込み単体）と v2（アンサンブル）を比較しました。

| 指標 | v1（埋め込みのみ） | v2（アンサンブル） |
|---|---|---|
| primary_topic 変更率 | — | **37.1%（371 件）** |
| BM25 有効件数 | — | 610 件（61.0%） |
| 埋め込みスコア平均 | 0.8119 | 0.8083 |
| BM25 スコア平均（有効のみ） | — | 0.9396 |

変更された 371 件はすべて BM25 が有効に機能したケースであり、日本語論文タイトルに含まれる英語技術用語・略語が Topic の keywords に一致したことで、より適切な Topic が選出されました。

### 変更例（抜粋）

| v1（埋め込みのみ） | v2（アンサンブル） | 評価 |
|---|---|---|
| Syntax, Semantics, Linguistic Variation | Linguistics, Language Diversity, and Identity | 改善 |
| Student Assessment and Feedback | Writing and Handwriting Education | 改善（日本語教育文脈） |
| Peacebuilding and International Security | Peace and Human Rights Education | 改善（IRDB 文脈に適合） |
| Diabetic Foot Ulcer Assessment | Sleep and related disorders | ⚠️ 要検討 |
| Geriatric Care and Nursing Homes | Dental Research and COVID-19 | ⚠️ 要検討 |

2 件については v1 の方が適切に見えるケースがあり、BM25 の重みを調整する余地があることも確認しています。

## 4. 今後のステップについてご意見をお伺いしたい点

以下の選択肢について、先生のご意見を伺えますと幸いです。

**オプション A：v2 で全 249 万件を再処理**
BM25 追加による改善が確認できれば、全件を v2 で再実行します。処理時間は v1 と同等（約 28 時間）の見込みです。

**オプション B：人手評価を先に実施**
変更のあった 371 件から 20〜30 件を抽出し、v1・v2 の Topic を専門家が採点してから、全件処理の判断を行います。

**オプション C：BM25 の重みを調整してから再試験**
現在の重み（0.75 / 0.25）を調整し（例：0.85 / 0.15）、過剰な変更を抑制した上で再度 1k 比較を行います。
